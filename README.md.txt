Site Web du Cabinet Dentaire BrightSmile

Bienvenue sur le projet du site web du Cabinet Dentaire BrightSmile !
Aperçu du Projet

Le site web du Cabinet Dentaire BrightSmile est un projet visant à créer une présence en ligne pour un cabinet dentaire fictif. Le site fournira aux patients potentiels des informations sur les services offerts par le cabinet, l'équipe médicale, les horaires d'ouverture et les moyens de contacter la clinique.
Objectifs du Projet

    Créer un site web professionnel et accueillant pour présenter le cabinet dentaire.
    Fournir des informations détaillées sur les services dentaires offerts, y compris la dentisterie générale, les soins orthodontiques, l'esthétique dentaire, etc.
    Présenter le personnel médical, y compris les dentistes, les hygiénistes dentaires et le personnel administratif.
    Fournir des informations sur les horaires d'ouverture et les moyens de contacter le cabinet, y compris le numéro de téléphone et l'adresse e-mail.

Technologies Utilisées

    HTML5 et CSS3 pour la structure et le style du site.
    JavaScript pour les fonctionnalités interactives telles que les formulaires de contact et les effets visuels.
    Framework Bootstrap pour le développement frontend, assurant la réactivité et la compatibilité multi-appareils.
    PHP pour la gestion des formulaires de contact et l'envoi de courriels.
    MySQL pour la gestion des données des patients et le stockage des formulaires de contact.

Structure des Fichiers

    index.html : Page d'accueil du site.
    style.css : style du site
